## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/react-hook-app-vite.git && cd react-hook-app-vite && npm install
```

### Executing program

```
npm start
```
